Overlay for any page

This page can only watch its own keystrokes. The overlay injects the full debugger (presses, raw stream, expandable event dumps, markers, locks) as a draggable, resizable panel onto whatever page it runs on, and reports the receiving target and defaultPrevented so you can see which element got the key and whether a handler ate it.

Option A · Bookmarklet

Drag this to your bookmarks bar, then click it on any site. Click it again to remove the overlay.

Option B · DevTools console

Paste this into the console on any page and press Enter. This is the version to use if the bookmarklet errors: it is plain JavaScript, not a javascript: URL.

Listens in the bubble phase: events stopped with stopPropagation() before reaching document will not appear. Sites with a strict Content-Security-Policy may block bookmarklets and console-pasted script. The overlay ignores keystrokes aimed at its own controls.
